# Search Test Automation

## Prerequisites:

* Chrome browser installed on your machine
* Chromedriver executable in the system's PATH or specify the path in the test code
* TestNG framework installed in your project

## Running the Tests:

1. Compile and run the test class using your preferred Java compiler and IDE.
2. The tests will be executed and the results will be displayed.

Note: Please ensure that you have the latest version of Selenium WebDriver and ChromeDriver installed on your machine for this test to work correctly.
