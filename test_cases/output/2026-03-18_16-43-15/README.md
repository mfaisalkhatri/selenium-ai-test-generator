# Running the Test

To run the test, follow these steps:

1. Make sure you have Java and Maven installed on your machine.
2. Clone this repository or download the files as a zip.
3. Navigate to the project directory in your terminal/command prompt.
4. Run the command `mvn test` to execute the test.

Note: Replace "/path/to/chromedriver" with the actual path to your ChromeDriver executable file.
